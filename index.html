New fedora-openstack project setup on github

  1. Fork $project to fedora-openstack org
  2. fedpkg clone $project && cd $project
  3. git remote add -f fedora-openstack git@github.com:fedora-openstack/$project.git
  4. git checkout fedora-openstack/master
  5. git checkout -b master-patches # New local branch
  6. git push --set-upstream fedora-openstack master-patches # New upstream branch
  7. On github set master-patches branch to default 8 git push fedora-openstack :master # Delete upstream master branch

setup local repo (if you someone else set it up on github)

  1. fedpkg clone $project && cd $project
  2. git remote add -f fedora-openstack git@github.com:fedora-openstack/$project.git
  3. git branch master-patches fedora-openstack/master-patches

finish setup of local repo

  1. git remote add -f openstack git://github.com/openstack/$project.git
  2. git remote add -f fedora-smokestack git@github.com:fedora-openstack/openstack-$project.git
